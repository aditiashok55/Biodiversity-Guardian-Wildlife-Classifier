## Project Overview

Manual identification of animal species from images is time-intensive, requires domain expertise, and does not scale to the volume of imagery generated by modern camera trap networks and biodiversity monitoring systems. This project develops an end-to-end deep learning solution for automated wildlife classification, targeting 27 ecologically significant species.

The classifier leverages **EfficientNetB3** pre-trained on ImageNet as a convolutional backbone, combined with a two-phase transfer learning strategy (frozen feature extraction followed by selective fine-tuning). The system is designed to support conservation practitioners, ecological researchers, and biodiversity management programmes requiring scalable, reliable automated species identification.

## Exploratory Data Analysis

A preliminary data exploration was conducted on an auxiliary wildlife dataset to assess class distribution, image quality variance, and inter-class visual similarity. Key findings from this analysis informed augmentation strategy selection and class weighting decisions for training:

- **Class imbalance** was observed across several species groups, motivating the use of geometric augmentation to artificially balance representation of underrepresented classes.
- **Intra-class visual variance** (e.g., lighting conditions, occlusion, background clutter) confirmed the need for a robust pretrained backbone rather than training from scratch.
- **Inter-class confusion candidates** (e.g., Gorilla/Chimpanzee, Seal/Otter) were identified as likely sources of classification error, informing post-hoc analysis of the confusion matrix.

Visualisations produced during this phase include class frequency distributions, sample image grids per class, and pixel intensity histograms.

## Model Architecture

### Backbone

```
EfficientNetB3 (pretrained on ImageNet)
  Input shape : (224, 224, 3)
  include_top : False
  Output      : Feature map → GlobalAveragePooling2D
```

EfficientNetB3 applies compound scaling (depth, width, resolution) and uses MBConv blocks with Squeeze-and-Excitation, providing an efficient accuracy-to-parameter tradeoff suitable for fine-grained visual classification.

### Classification Head

```
GlobalAveragePooling2D
    ↓
Dense(256, activation='relu')
    ↓
Dropout(0.4)
    ↓
Dense(27, activation='softmax')
```

### Parameter Summary

| Component | Status (Phase 1) | Status (Phase 2) |
|---|---|---|
| EfficientNetB3 base | Frozen | Top 80 layers trainable |
| Classification head | Trainable | Trainable |

## Training Pipeline

### Phase 1 — Feature Extraction (Frozen Backbone)

The base model weights are held fixed; only the classification head is trained. This prevents catastrophic forgetting while adapting the head to the target distribution.

```python
optimizer : Adam(lr=3e-4)
loss      : categorical_crossentropy
epochs    : up to 15 (early stopping)
```

**Callbacks:**
- `EarlyStopping(patience=5, restore_best_weights=True)`
- `ReduceLROnPlateau(patience=3, factor=0.3, min_lr=1e-6)`
- `ModelCheckpoint("best_model.keras", save_best_only=True)`

### Phase 2 — Selective Fine-Tuning

The top 80 layers of the backbone are unfrozen and trained at a reduced learning rate to adapt higher-level feature representations to the wildlife domain while preserving low-level edge/texture detectors.

```python
base_model.trainable = True
# Freeze all layers except the top 80
for layer in base_model.layers[:-80]:
    layer.trainable = False

optimizer : Adam(lr=1e-4)
loss      : categorical_crossentropy
epochs    : up to 40 (early stopping)
```

**Phase 2 Training Log (excerpt):**

| Epoch | Train Acc | Val Acc | Val Loss | LR |
|---|---|---|---|---|
| 1 | 92.96% | 97.94% | 0.0793 | 1e-4 |
| 2 | 97.07% | 97.94% | 0.0759 | 1e-4 |
| 3 | 97.64% | — | 0.1038 | 1e-4 |

*Training was halted early once convergence was confirmed (minimal marginal gain, low validation loss).*

## Installation & Usage

### Requirements

```bash
pip install tensorflow>=2.10
```

### Inference Example

```python
import tensorflow as tf
import numpy as np
from tensorflow.keras.applications.efficientnet import preprocess_input
from tensorflow.keras.preprocessing import image

KEEP_CLASSES = [
    "chimpanzee", "dolphin", "eagle", "elephant", "flamingo",
    "gorilla", "hornbill", "koala", "leopard", "lion",
    "octopus", "okapi", "orangutan", "otter", "panda",
    "parrot", "penguin", "rhinoceros", "seal", "seahorse",
    "shark", "tiger", "turtle", "whale", "zebra",
    "hippopotamus", "wolf"
]

model = tf.keras.models.load_model("best_model.keras")

img = image.load_img("sample.jpg", target_size=(224, 224))
x   = image.img_to_array(img)
x   = preprocess_input(np.expand_dims(x, axis=0))

preds      = model.predict(x)
class_idx  = np.argmax(preds, axis=1)[0]
confidence = preds[0][class_idx]

print(f"Predicted: {KEEP_CLASSES[class_idx]} ({confidence:.2%})")
```
